# Fruit-recognition-using-vgg-16
Overview

This project implements a Fruit Recognition System using custom architecture based on the VGG16 Convolutional Neural Network (CNN). The model is trained on fruit images and can classify different fruit categories with high accuracy.
Features
Fruit image classification using custom architecture based on VGG16
Transfer learning with pre-trained ImageNet weights
Image preprocessing and augmentation
Training and validation performance visualization
Prediction on custom fruit images


Dataset

The dataset contains images of various fruit categories.

Example classes:

Apple
Banana
Orange
Mango
Grapes
Pineapple
