====================================================

PROPOSED CONTRIBUTIONS

1. Competitive Activation Function (CAF)

   - Custom trainable activation function
   - Designed as an alternative to ReLU
   - Introduces adaptive feature scaling
   - Improves feature representation capability

2. Advanced Fruit Discrimination Loss (FDL)

   - Three-term custom loss function

   Components:

   a) Cross Entropy Term

   b) Margin Separation Term

   c) Feature Similarity Penalty Term

   Purpose:

   - Better inter-class separation
   - Improved discrimination between visually similar fruits

3. FruitNet Architecture

   Input
      |
   Multi-Scale Block
   (3x3 + 5x5 + 7x7)
      |
   Attention Block
      |
   VGG16 Backbone
      |
   Custom Embedding
      |
   Classifier

4. Three-Phase Training Strategy

   Phase 1:
   Classifier Training

   Phase 2:
   Partial Fine Tuning

   Phase 3:
   Full Fine Tuning

KEY FINDINGS

1. Multi-Scale features improve fruit representation.

2. Attention mechanisms help focus on relevant fruit regions.

3. CAF successfully learns adaptive activations.

4. Advanced FDL improves class discrimination.

5. FruitNet achieves strong performance on a highly imbalanced dataset.

6. GradCAM analysis confirms focus on fruit-specific regions.

====================================================

LIMITATIONS

1. Training requires GPU acceleration.

2. Dataset contains background similarities.

3. CAF introduces additional computational cost.

4. Performance evaluated on a single dataset.
